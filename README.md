# issue-swarm-skill

A skill for `opencode` that spawns parallel coding agents to work on multiple GitHub issues using git worktrees.

## Installation

Clone this repository into your agents skills directory:

```bash
mkdir -p ~/.config/agents/skills
git clone https://github.com/lsj5031/issue-swarm-skill ~/.config/agents/skills/issue-swarm
```
